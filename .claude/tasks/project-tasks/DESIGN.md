> Reference: premium law firm websites, Notion AI, Perplexity Pro.

## Typography

- **Display / Headings:** Playfair Display (`--font-display`) — brand name, conversation title, AI card header label, hero H1
- **Body:** Inter (`--font-sans`) — all paragraph text, UI labels, input
- **Mono:** JetBrains Mono (`--font-code`) — IDs, badges, code spans
- **Body size:** 13.5px, `line-height: 1.85` in AI cards; 13px elsewhere
- **Messages:** 13.5px, `line-height: 1.85`

## AI Response Card (key differentiator)

```
┌───┬──────────────────────────────────────────────────────┐
│LS │ LegalSathi                    [2 sources] [Copy]     │ ← header (divider below)
│   ├──────────────────────────────────────────────────────┤
│   │ [Response text, 13.5px Inter, leading-[1.85]]        │
│   │                                                      │
│   │ ─────────────────── (if sources exist) ───────────── │
│   │ Referenced: 📄 Document A   📄 Document B            │
│   │                                                      │
│   │ ─────────────── (if suggestions exist) ────────────  │
│   │ ⚡ YOU MIGHT ALSO ASK                                 │
│   │ [What are my rights?]  [How to file?]  [Penalties?]  │
└───┴──────────────────────────────────────────────────────┘
```

- Left border: `3px solid #1E2E4F`
- Background: `#FFFFFF`
- Header + card divider: `1px solid #F0EBE3`
- Copy button + source count badge: visible only on `group-hover`
- Avatar: `32px`, `rounded-xl`, navy bg

---

## Follow-up Suggestions (key differentiator)

Generated by `claude-haiku-4-5` after each response (3 questions, ≤9 words each).  
Sent as `{ suggestions: string[] }` SSE event after `{ sources }`.  
Rendered as pill chips at the bottom of the AI card.  
Clicking a chip calls `sendMessage(question)` directly — no manual send needed.

---

## User Message Bubble

- Shape: `border-radius: 18px 18px 4px 18px` — pill except bottom-right
- Background: `#1E2E4F` (navy)
- Text: `#EEE9DF` (warm white)
- Max-width: 68%
- Right-aligned, no avatar

---

## Copy Button

- Appears on `group-hover` of the AI card header (opacity transition)
- Shows `<IconCopy> Copy` by default
- On success: flips to `<IconCheck> Copied` in green for 2 seconds
- Uses `navigator.clipboard.writeText()`

---

## Shadows

- AI card: `shadow-sm` (`0 1px 2px rgba(0,0,0,0.05)`)
- User bubble: `shadow-sm`
- Avatars: `shadow-sm`
- Input focus: `shadow-sm` + `border-app-border-strong`

---

## Animations

- Hover transitions: `150ms`
- Copy button: `opacity 0 → 1` on group-hover
- Suggestion chip hover: bg + text color swap (inline `onMouseEnter/Leave`)
- Thinking dots: `dot-pulse` opacity pulse only
- No transforms, no scale, no spring

---

## Explicitly Banned

- No gradient backgrounds
- No glassmorphism
- No colored glows
- No uppercase in main content (SMALL CAPS in suggestion section header only)
- No fake metadata (latency, build numbers, version strings)
- No hero sections inside the active chat view
- No dramatic drop shadows in main content
- No `rounded-full` on non-pill elements
