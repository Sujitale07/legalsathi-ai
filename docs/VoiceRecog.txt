# TASK: Create Standalone Voice Assistant Route (MVP)

## Goal

Create a completely separate route for a Voice Assistant MVP.

This route should be isolated from the existing LegalSathi features.

Do NOT integrate with RAG, legal datasets, lawyer finder, or contract review yet.

The goal is only:

* Start voice conversation
* Speak to Gemini
* Receive voice responses
* Maintain conversation context
* Simple UI
* Working demo

---

# Route Structure

Create:

```txt

```

# Buttons

## Start Conversation

When clicked:

* Connect to Gemini Live API
* Create voice session
* Request microphone permission
* Start listening

Disable button while connected.

---

## End Conversation

When clicked:

* Disconnect Gemini session
* Stop microphone
* Clear buffers

Disable button when not connected.

```txt

```
